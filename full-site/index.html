Pin the Web: site with web manifest

This is a complete site made up of multiple pages, associated together and given shared attributes using Firefox OS's new support for the W3C Manifest for a web application specification. You can see other examples at our RDFa and Basic landing page pages.

You can find out more at Pin the Web on MDN.
